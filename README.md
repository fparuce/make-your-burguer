# 3_make_your_burguer

Projeto para estudo de JavaScript com framework VueJS.
Utilizando json-server como pseudo-backend.

## Project setup
```
npm install
npm install json-server
```

### Compiles and hot-reloads for development
```
npm run backend
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
